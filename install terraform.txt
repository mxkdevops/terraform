

1. Download the appropriate Terraform binary package for the provided lab server VM (Linux 64-bit) using the wget command:

  wget -c https://releases.hashicorp.com/terraform/0.13.4/terraform_0.13.4_linux_amd64.zip


# wget command to retrive files from the web 

# -c options to conitunue downloading an incomplete file if its interrupted 

#  https://releases.hashicorp.com/terraform/0.13.4/terraform_0.13.4_linux_amd64.zip: The URL of the Terraform binary for 
   version 0.13.4 and the Linux AMD64 architecture in ZIP format.

# It will download binary zip file in the current directory 

# Need to unzip file and place the binary in a directory included your system PATH


2. Unzip the downloaded ZIP file:

unzip terraform_0.13.4_linux_amd64.zip

3. Move the Terraform binary to a directory in your PATH, such as /usr/local/bin/

sudo mv terraform /usr/local/bin/

4. Make the binary executable 

sudo chmod +x /usr/local/bin/terraform

-o option with wget
wget -c -o terraform_0.13.4_linux_amd64.zip https://releases.hashicorp.com/terraform/0.13.4/terraform_0.13.4_linux_amd64.zip

-o terraform_0.13.4_linux_amd64.zip specifies the output filename as terraform_0.13.4_linux_amd64.zip.
